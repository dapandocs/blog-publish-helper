#### 博客直达小助手
这是一款一键分发博客的浏览器插件工具，支持多个博客网站，包括掘金、知乎、CSDN、简书、博客园、开源中国、SegmentFault等等。使用它可以轻松将您的博客文章发布到多个平台，节省您的时间和精力。
